# CAP AUTO BACK-END REST API

## Introduction

> CAP AUTO est une application WEB qui répond à un besoin client, elle a pour de buts de gérer une auto-école.

## Techno

- JAVA
- Spring boot
- JPA

## Requirements

For building and running the application you need:

- [JDK 1.8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
- [Maven 3](https://maven.apache.org)

## Running the application locally

There are several ways to run a Spring Boot application on your local machine. One way is to execute the `main` method in the `com.aston.capauto.CapAutoApplication` class from your IDE.

Alternatively you can use the [Spring Boot Maven plugin](https://docs.spring.io/spring-boot/docs/current/reference/html/build-tool-plugins-maven-plugin.html) like so:

```shell
mvn spring-boot:run
```
